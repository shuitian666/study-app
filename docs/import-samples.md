# Import Samples

用于固定导入解析的最小回归样本集，方便每次调整 `ImportKnowledge` 解析器后快速回归。

推荐回归方式：

1. 运行 `npm run import:samples`
2. 到 `output/import-samples/` 查看生成的样本文件
3. 在导入页逐个拖入或粘贴样本
4. 对照下面的“预期模式 / 预期卡片数 / 预期提示”检查结果

说明：

- “卡片数量”指最终应导入的 `KnowledgePoint` 数量
- “题目数量”是附加参考项，方便检查知识点+题目关联是否回归
- 置信度与 warning 文案允许微调，但含义不应跑偏

## 样本总览

| S01 | 标准 Q/A | `standard-qa.txt` | `qa` | 2 | 0 | 无 warning，置信度高 |
| S02 | 标准知识点 | `standard-knowledge.txt` | `knowledge` | 2 | 0 | 无 warning，置信度高 |

| S05 | 半残缺 Q/A | `incomplete-qa.txt` | `qa` | 2 | 0 | 提示检测到 Q 缺少 A |
| S06 | 半残缺题目解析 | `incomplete-question-explanation.txt` | `knowledge` | 1 | 1 | 提示检测到【题目】缺少【解析】 |

### S03 - 知识点+题目+解析混合

- 预期模式：`knowledge`
- 预期卡片数：`2`
- 预期题目数：`2`

```txt
【知识点】牛顿第一定律
内容：物体在不受外力时保持静止或匀速直线运动状态。
【题目】题干：以下哪项最符合牛顿第一定律？
A. 受力越大，速度一定越大
B. 不受外力时物体保持原有运动状态
C. 所有物体都会自动停止
答案：B
【解析】牛顿第一定律强调的是“保持原有状态”。

【知识点】氧化还原反应
内容：反应过程中发生电子转移或元素化合价变化。
【题目】题干：以下哪项属于氧化还原反应的核心特征？
A. 颜色变化
B. 状态变化
C. 电子转移
答案：C
【解析】判断氧化还原反应的核心标准是是否存在电子转移。
```

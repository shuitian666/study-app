## 竞争格局分析：TRAE 该如何定位

### 方案对比

#### 方案 A：对标墨墨（内容精美 + 订阅）

```
优势：
✅ 已经有医学垂直内容
✅ 可对标的成熟商业模式
✅ 市场需求明确

劣势：
❌ 竞争对手已占据单词领域
❌ 医学领域更小众，变现困难

建议价格：¥98-168/年
重点投资：内容质量、周报系统、卡片市场
```

#### 方案 B：对标高途（垂直领域 + 师资）

```
优势：
✅ 医学是高价值领域
✅ 可以吸引医学教授入驻
✅ 高客单价（课程可卖 ¥499-1999）

劣势：
❌ 需要强大的网红/名师资源
❌ 重模式，需要大量人工

建议投资：邀请医学教授、录制专家课程、建立师资库
重点周期：12-18 个月
```

#### 方案 C：对标 Anki（开源生态 + API）

```
优势：
✅ 符合你的技术基因
✅ 可以吸引开发者和高端用户
✅ 网络效应强

劣势：
❌ 变现困难
❌ 需要建立强大的生态

建议方向：发布 API + SDK，让第三方开发者创建医学卡片库
```

**推荐：方案 A + B 的融合**
- 短期（6个月）：做好方案 A（内容质量 + 周报系统）
- 中期（12个月）：邀请医学教授，推出"名师讲解"课程
- 长期（24个月）：建立完整的医学教育生态

export async function exportAsCSV(learningState) {
  const rows = learningState.knowledgePoints.map(kp => [
    kp.id,
    kp.name,
    kp.explanation,
    kp.proficiency,
    kp.reviewCount,
    kp.lastReviewedAt
  ]);

// 导出 Anki
export async function exportAsAnki(learningState) {
  return learningState.knowledgePoints.map(kp => ({
    Front: kp.name,
    Back: kp.explanation,

  }));
}
